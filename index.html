Hi! I'm Ricky and I'm a bit obsessed with WordPress, lately. Somehow it's become the common thread that runs through all the work I do. At my day job, I fulfill client requirements by developing WordPress themes. In the evenings and on the weekends, I organize meetups through WordPress Frederick. I also like to browse the official WordPress support forum to help people out on there.

WordPress empowers me to build cool stuff within a predictable structure. It enables my clients to work with their own content in a familiar dashboard. As a web enthusiast, I like to try new frameworks and software because it's a lot of fun to tinker, to approach problems from different angles. But I always happily come back to WordPress.

My Journey

Working with the various web languages (HTML, CSS, JavaScript, PHP) makes me feel like a wizard, honestly. When I was 12 years old I discovered that I could write choose-your-own-adventure games on my TI-86 graphing calculator with TI-BASIC. Decades later, I'm still endlessly fascinated with giving computers instructions to carry out. Coding both sates and fuels my curiosity, like a perpetual motion machine that keeps me moving forward.

Shout-out to The Odin Project for helping me turn my hobby into a career!

Favorite Tools

Here are some of the tools that I currently enjoy using:

  • Local spins up WordPress sites in seconds so I can experiment freely.
  • VS Code with Vim emulation gets me right into that brain-to-code flow state.
  • Colemak is an alternative keyboard layout that makes touch typing feel effortless.
  • AutoHotkey can be used to automate anything, which is a fun challenge on its own!
  • uBlock Origin and Tampermonkey give me granular power over the sites I visit.

This Site

WordPress is my hammer and I admit it can make every website project look like a nail. But a database-driven content management system is overkill for a site like this one. To introduce myself and share some interesting links, I really only need a basic set of HTML pages. So that's what I've put together here. I dropped in a minimalist stylesheet from Simple.css to keep the content pleasantly readable. For me, the most attractive advantage of this tiny static site is I can host it anywhere. I chose GitHub Pages.

As I write this, I'm manually moving markup tags out of my way to get my thoughts onto the page. Tight integration with Jekyll is a major feature of GitHub Pages, which gives me a great excuse to try that soon!

Thanks for stopping by! This site is styled with Simple.css and hosted on GitHub Pages.
